# 示例：腾讯 QQ 部门用户产品场景题

这个示例展示 `pm-scenario-coach` 的典型使用方式：用户只给公司和方向，skill 自动出题，并按照 STAR 法则轻量拆解。

## 用户输入

```text
$pm-scenario-coach 我想练腾讯 QQ 部门产品经理
```

## Skill 出题

```text
好，我们来练一题：腾讯 QQ / 用户产品。

场景题：
假设你是 QQ 产品经理，发现很多新用户加入 QQ 频道或兴趣群后，只浏览不发言，24 小时内首次互动率偏低。你会如何提升新用户的首次有效互动率？

我们按照 STAR 法则来分析，也就是：场景 × 目的 × 行为 × 结果。
S：场景，判断这道题发生在什么真实业务情境里。
T：目的，判断这道题真正要解决什么问题。
A：行为，选择更符合公司调性的产品动作。
R：结果，判断用什么指标证明方案有效。

腾讯 QQ 更看重用户价值、关系链、低压力表达和长期社交体验。请你带着这个认知开始下面的思考。
```

## S：场景识别

```text
第一步：S，场景识别。
这一步不是让你急着想方案，而是先判断：用户为什么在这个场景下不行动。

从下面选项中，选出你认为合适的所有选项：
A. 主要问题是发言按钮不够显眼，所以应该优先改按钮颜色
B. 新用户刚加入兴趣群或频道，对氛围还不熟，不知道该不该发言
C. 只要用红包、等级、任务奖励刺激发言，就能解决互动率问题
D. 用户可能有表达意愿，但担心冷场、没人回应，或者显得突兀
E. 群/频道已有熟人或老用户互动，新用户需要一个低压力融入入口

直接回复你认为合适的所有选项。
```

用户：

```text
B D E
```

Skill：

```text
完全正确 抓到重点了 :) 这道题考查的是你能不能理解新用户进入 QQ 兴趣社交场景后的表达门槛。

B/D/E 都贴近腾讯 QQ 的产品语境：新用户不是完全不想表达，而是对氛围不熟、怕冷场、怕没人接。A 和 C 容易把题目带偏成入口优化或强刺激增长。

你这一步要学会的判断方法是：看到社交互动题，先判断用户是否有表达压力、关系链是否影响行为、方案是否会破坏社交氛围。
```

## T：目的判断

```text
第二步：T，目的判断。
从下面选项中，选出你认为合适的所有选项：
A. 最大化群消息量，越热闹越好
B. 帮助新用户更快理解群/频道氛围，降低融入门槛
C. 提升新用户首次有效互动率，让用户完成第一次低压力表达
D. 通过强提醒和任务奖励，让新用户必须发言
E. 提升长期留存和关系沉淀，而不是只看当天发言数

直接回复你认为合适的所有选项。
```

用户：

```text
B C E
```

Skill：

```text
完全正确 稳了 :D 这道题考查的是首次互动背后的融入门槛和长期关系沉淀。

B/C/E 构成了很完整的目标：先降低融入门槛，再完成首次低压力表达，最后看这次表达是否带来留存和关系沉淀。A 的“热闹”是虚荣指标，D 会破坏社交安全感。

你这一步要学会的判断方法是：腾讯 QQ 这类社交产品不能只追数量，要看互动是否自然、可被回应，并能沉淀关系。
```

## A：行为选择

```text
第三步：A，行为选择。
从下面选项中，选出你认为合适的所有选项：
A. 给所有新用户强制发言任务，不发言就无法继续浏览
B. 在新用户进入后展示群/频道氛围卡片，让他快速知道这里聊什么、怎么聊
C. 给新用户提供低压力互动入口，比如投票、表情回应、话题接龙、轻量自我介绍模板
D. 用红包、积分、等级奖励刺激用户大量发言
E. 识别高活跃或友好的老用户，引导他们回应新用户的首次表达

直接回复你认为合适的所有选项。
```

用户：

```text
B C E
```

Skill：

```text
完全正确 :D 这道题考查的是你能不能把“提升互动”拆成低压力表达、氛围理解和关系承接。

B/C/E 的组合很完整：先让新用户知道这里是什么氛围，再给一个不尴尬的轻互动入口，最后通过老用户回应承接第一次表达。A 和 D 都过度强刺激，容易伤害社交体验。

你这一步要学会的判断方法是：社交产品里的增长动作要降低心理门槛，而不是制造压力。
```

## R：结果判断

```text
第四步：R，结果判断。
从下面选项中，选出你认为合适的所有选项：
A. 红包/积分任务完成率
B. 新用户 24 小时内首次有效互动率
C. 新用户互动后收到回复或反馈的比例
D. 群/频道总消息量最大化
E. 新用户首次互动后的 7 日留存率

直接回复你认为合适的所有选项。
```

用户：

```text
B C E
```

Skill：

```text
完全正确 抓到重点了 :) 这道题考查的是你能不能用指标证明“互动质量”和“长期关系沉淀”，而不是只看群里热不热闹。

B 是主指标，C 是过程质量指标，E 是长期结果指标。A 和 D 都容易跑偏：任务完成不等于真实融入，总消息量也不等于有效互动。

你这一步要学会的判断方法是：社交题的结果指标要同时看首次行为、被回应质量和后续留存。
```

## 最终逐字稿

```text
这道题你真正要学会的 3 条判断规则：
1. QQ 社交题先看表达压力，不要急着上强刺激。
2. 首次互动不是消息量问题，而是低压力融入和被回应问题。
3. 指标要同时看行为发生、互动质量和后续留存。

这道题的逐字稿可以这样说：

如果我来做这道题，我会先把问题定义为：新用户加入 QQ 兴趣群或频道后，并不是完全没有表达意愿，而是对群体氛围不熟，担心发言突兀、没人回应或者冷场，所以会先选择观望。

因此目标不是简单提升群消息量，而是帮助新用户在低压力、可被回应的场景下完成第一次有效互动，并为后续留存和关系沉淀打基础。

具体方案上，我会分三步做。第一，降低新用户第一次表达的门槛，比如提供投票、表情回应、话题接龙、轻量自我介绍模板。第二，在用户进入群或频道后，用氛围卡片告诉他这里主要聊什么、适合怎么参与。第三，识别高活跃或友好的老用户，引导他们回应新用户的第一次表达，避免新用户发完没人接。

指标上，我会把新用户 24 小时内首次有效互动率作为主指标，同时看首次互动后收到回复或反馈的比例，再看首次互动后的 7 日留存率。护栏上，我也会关注低质刷屏、举报率、退群率，避免为了提升互动而破坏群氛围。

整体上，这个方案不是为了让群变得更吵，而是让新用户更自然地融入 QQ 的兴趣社交关系里。

你想继续怎么练？
A. 再练一题
B. 复盘这题，看每一步为什么这么选
```
